# Oficina de Integração: Engenharia Reversa e Sustentabilidade

Repositório destinado ao desenvolvimento do projeto da disciplina de **Oficina de Integração**, com foco em práticas de engenharia reversa aplicadas à sustentabilidade tecnológica.

---

## 📌 Visão Geral
O projeto consiste na análise técnica de um sistema/produto existente para compreender sua arquitetura, funcionamento e viabilidade de reaproveitamento, alinhando inovação tecnológica com a redução de resíduos eletrônicos.

### Temáticas Centrais:
1. **Engenharia Reversa:** Desconstrução e mapeamento de componentes de hardware e software.
2. **Sustentabilidade:** Extensão do ciclo de vida útil ou proposta de reciclagem eficiente.
